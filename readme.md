## Laravel Blog

Building a blog from scratch. Experimenting with SQLite, Gulp, and Bootstrap. Maybe a little angular.


### Instructions for dev

1. `composer update`
2. `bower install`
3. `npm install`
4. `gulp less`
5. `gulp concat`
6. `gulp css`
7. `gulp uglify`	
8. `gulp` (to watch)
